Some HTML Tags!

HTML TAG: USAGE
<a> Used to link an item.
<article> Defines an article
<aside> Defines a part of an article that is related, but not necesserily dominant.
<body> Defines the visible part of the document.
<div> Defines a part of the document.
<em> Italic writing
<embed> Used to include media files in web-sites.
<figcaption> Used to include a caption for the figure.
<figure> Used to add images to websites.
